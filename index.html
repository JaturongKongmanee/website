Research

I'm interested in machine learning, software engineering, and building tools and theory to improve human-ML interactions. Representative papers are highlighted.

  • Current Project: I am developing tools and machine learning algorithms under active learning paradigms for Sun Life. The goal is to efficiently utilize the available unlabeled data and human experts to mitigate data exfiltration, which is an important problem in cybersecurity.

Multi-objective Coevolution and Decision-making for Cooperative and Competitive Environments
Anirudh Suresh, Jaturong Kongmanee, Kalyanmoy Deb, Vishnu Naresh Boddeti

CEC 2021: IEEE Congress on Evolutionary Computation
[ paper ]

We propose a proof-of-principle multi-objective co-evolutionary (MOCoEv) study, which is of great interest in cooperative and competing games and search tasks in which multiple agents having different interests are in play.

Fine-Tuning A Lightweight Convolutional Neural Networks for COVID-19 Diagnosis
Jaturong Kongmanee, Thanyathorn Thanapattheerakul

CSBio 2020: International Conference on Computational Systems-Biology and Bioinformatics
[ paper ]

We compare and optimize the performance of the deep neural network-based image classifiers for an automatic COVID-19 diagnosis using limited chest x-ray image datasets provided by Deep Learning and Artificial Intelligence Summer School 3 (DLAI3).

Securing smart contracts in blockchain
Jaturong Kongmanee, Phongphun Kijsanayothin, Rattikorn Hewett

ASE 2019: IEEE/ACM International Conference on Automated Software Engineering
[ paper ]

We present a general technique for building the core functional models applicable for model checking to identify all possible executions that lead to security breaches. We illustrate and evaluate the technique with a widely used example of a smart contract in a financial system along with experimental results using a well-known model checker, NuSMV, in various scenarios.

Network-based Visualization Tool for Analyzing Gene Expression Data
Thanyathorn Thanapattheerakul Jaturong Kongmanee, Jonathan H. Chan

VINCI 2017: International Symposium on Visual Information Communication and Interaction
[ paper ]

We present an integrated network-based visualization tool for gene expression analysis. Specifically, this web-based tool allows researchers to visualize and search gene pathways to gain a better understanding of disease biomarkers.
